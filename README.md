# 💪 FZ Fitness

A modern and responsive gym/fitness landing page built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- Modern dark gym theme
- Responsive design for all devices
- Smooth navigation menu
- Hero section with background image
- Programs and pricing sections
- Contact form UI
- Clean and minimal layout

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
